Ce que je propose

  • Conception de sites web embarqués pour le pilotage et la supervision d’équipements
  • Mise en œuvre de solutions de sécurité pour objets connectés (TLS, chiffrement, authentification)
  • Développement d’interfaces techniques sur microcontrôleurs (HMI) ou PC (Qt, NW.js, modèle 3D)
  • Programmation firmware bas niveau sur MCU 32 bits (C/C++, FreeRTOS, AzureRTOS – STM32, PIC32, RA, S7G2)
  • Intégration de modems cellulaires, Wi-Fi, radio et protocoles réseau sécurisés (MQTT, LoRa, HTTPS)
  • Support pour validation CE, analyse critique de bugs, et fiabilisation de produits

Exemples de réalisations

Librairie SSL/TLS over UART – PIC32MZ + Modem Heracles 224G
  • Objectif : Sécuriser les échanges série avec un modem 4G limité à TLS 1.0
  • Réalisation : Implémentation TLS 1.2/1.3, gestion mémoire, handshake, certificats embarqués
  • Impact : Communication sécurisée via HTTPS/MQTTS pleinement opérationnelle
Réseau de capteurs industriels LoRa
  • Objectif : Créer un réseau robuste dans un environnement à forte contrainte
  • Réalisation : Bare metal + protocole propriétaire, gestion basse conso
  • Impact : Longue autonomie, fiabilité validée sur site client
Interface de test PC pour banc de validation (Qt + USB)
  • Objectif : Piloter les tests électroniques depuis un PC
  • Réalisation : UI en Qt, logs série, automatisation partielle
  • Impact : Gain de temps x3 sur la phase de validation
